# Employee Analytics & Workforce Insights

## Objective

The goal of this project was to analyze employee data to understand workforce trends, identify factors influencing employee turnover, and provide insights to improve retention and workforce planning.

---

## Dataset

The dataset contains employee-level information including:

* Age
* Gender
* Education level
* City
* Payment tier
* Work experience
* Bench status
* Joining year
* Attrition status (LeaveOrNot)

---

## Tools Used

* SQL  - Data cleaning and analysis
* Tableau - Data visualization and dashboard creation

---

## Project Structure

* `employee_analysis.sql` - Data cleaning and analysis queries
* Tableau Dashboard - Interactive visualization of workforce trends

---

## Data Preparation

* Checked and handled missing values across all columns
* Converted categorical values (e.g., EverBenched → binary format)
* Ensured data consistency for accurate analysis

---

## Exploratory Data Analysis

The analysis focused on answering key business questions:

* What is the overall employee turnover rate?
* Which factors influence employee attrition?
* How does turnover vary by education, city, and gender?
* What are the hiring trends over time?
* What percentage of employees were benched?

---

## Key Insights

* The overall **employee turnover rate is significant**, indicating potential retention challenges.

* Employees in certain **education categories show higher turnover**, suggesting mismatches in role expectations or career growth opportunities.

* **Turnover varies across cities**, indicating location-specific workforce challenges.

* A notable percentage of employees were **previously benched**, which may contribute to dissatisfaction and eventual attrition.

* Hiring trends show fluctuations over the years, with peaks followed by declines, indicating inconsistent workforce planning.

* Gender-based analysis shows **differences in experience levels and turnover**, highlighting potential diversity or engagement gaps.

---

## Recommendations

* Improve **employee retention strategies**, especially for high-turnover education groups.

* Reduce bench time by improving **project allocation and workforce utilization**.

* Develop **location-specific HR strategies** to address city-level challenges.

* Implement **consistent hiring strategies** to avoid workforce instability.

* Enhance **employee engagement programs** to reduce attrition risk.

---

## Dashboard

The Tableau dashboard provides an interactive view of:

* Hiring trends over time

---

## Conclusion

This project demonstrates how employee data can be used to uncover workforce trends and support HR decision-making. By combining SQL analysis with Tableau visualization, the project provides actionable insights for improving employee retention and organizational planning.
